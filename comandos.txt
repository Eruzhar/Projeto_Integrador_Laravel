--comando github para baixar o projeto

Comando: git clone https://github.com/Eruzhar/Projeto_Integrador_Laravel.git
Descrição: Copia um repositório Git existente para um novo diretório no computador local.


--comandos github para atualizar o repositorio baixado.

Comando: git pull origin master
Descrição: Baixa e mescla alterações do repositório remoto para o repositório local. 



--comandos github para remover arquivos do projetos

comando: git rm
Descrição: Remove arquivos de um repositório Git. Ele pode remover arquivos individuais ou uma coleção de arquivos. 



--comandos github para subir o projeto

Comando: git add .
Descrição: Adiciona alterações feitas em arquivos para o próximo commit no Git.

Comando: git commit -m "comentário para facilitar o entendimento do commit"
Descrição: Leva as mudanças de um ambiente local para o repositório no git, permitindo ainda a inserção de uma mensagem descritiva

Comando: git push origin master
Descrição: Envia as alterações feitas no branch master local para o branch master remoto.



--comandos de PHP para fazer o projeto funcionar 

Comando: composer install
Descrição: Lê o arquivo composer.json do diretório atual, resolve as dependências e as instala no vendor. 

Comando: npm install
Descrição: É usado para instalar pacotes e bibliotecas JavaScript em projetos Node.js

Comando: php artisan migrate
Descrição: Publica o esquema no banco de dados e gera uma tabela. 

Comando: php artisan serve
Descrição: Inicia um servidor de desenvolvimento embutido no Laravel

Comando: php artisan key:generate
Descrição: Gera uma nova chave de aplicativo para o Laravel.


php artisan storage:link
